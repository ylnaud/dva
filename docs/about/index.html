Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento escrito en HTML o XML. Para más información, visita MDN Web Docs.

Propiedades CSS

CSS ofrece una amplia gama de propiedades para controlar el estilo de los elementos. Puedes encontrar una referencia completa en W3Schools.
